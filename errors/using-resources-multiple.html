Using with multiple resources

Using.Manager handles multiple resources safely

Code Comparison
✕ Nested try/finally 
val in = new FileInputStream("in.txt")
try {
  val out = new FileOutputStream("out.txt")
  try {
    // use in and out
  } finally out.close()
} finally in.close()
✓ Using.Manager 
import scala.util.Using

Using.Manager { use =>
  val in = use(new FileInputStream("in.txt"))
  val out = use(new FileOutputStream("out.txt"))
  // All resources closed automatically
  copyStream(in, out)
}
Old Approach
Nested try/finally
Modern Approach
Using.Manager
Since Scala
2.13
Difficulty
Intermediate
How It Works

Manager ensures all resources are closed even if some acquisitions fail.
