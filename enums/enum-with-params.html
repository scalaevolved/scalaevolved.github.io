Enums with parameters

Enums support parameters just like case classes

Code Comparison
✕ Scala 2 
sealed trait HttpStatus
case class Ok(body: String) extends HttpStatus
case class NotFound(path: String) extends HttpStatus
case object ServerError extends HttpStatus
✓ Scala 3 
enum HttpStatus:
  case Ok(body: String)
  case NotFound(path: String)
  case ServerError

How It Works

Scala 3 enums can have parameters, combining the conciseness of enums with the power of ADTs.
