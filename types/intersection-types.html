Intersection types

Use & for intersection types with cleaner syntax

Code Comparison
✕ Scala 2 
trait Resetable {
  def reset(): Unit
}
trait Closeable {
  def close(): Unit
}

def cleanup[T <: Resetable with Closeable]
  (resource: T): Unit = {
  resource.reset()
  resource.close()
}
✓ Scala 3 
trait Resetable:
  def reset(): Unit

trait Closeable:
  def close(): Unit

def cleanup(resource: Resetable & Closeable) =
  resource.reset()
  resource.close()

How It Works

Intersection types use the & operator, making the intent clearer and syntax more consistent with union types.
