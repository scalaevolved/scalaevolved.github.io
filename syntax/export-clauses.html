Export clauses

Export members from another object without delegation boilerplate

Code Comparison
✕ Scala 2 
class Wrapper(val inner: Service) {
  def method1 = inner.method1
  def method2 = inner.method2
  def method3 = inner.method3
}
✓ Scala 3 
class Wrapper(val inner: Service):
  export inner.*

How It Works

Export clauses automatically forward members, eliminating manual delegation code.
